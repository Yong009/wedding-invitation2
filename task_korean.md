# 결혼식 초대장 스크롤 문제 해결 과제

사용자가 웹사이트(https://yong009.github.io/wedding-invitation2/) 접속 시 스크롤에 문제가 있다고 보고함.

## 문제 분석 결과
1. **데스크톱 '데드존(Dead Zone)' 문제**: 
   - 콘텐츠가 중앙에 고정(max-width: 480px)되어 있고, `body`에 `overflow: hidden`이 있거나 스크롤이 해당 영역 내에서만 작동하게 하여 사용자가 좌우 여백에서 휠을 돌릴 때 반응이 없음.
2. **초기 로딩 시 레이아웃 시프트(Layout Shift)**:
   - 메인 이미지가 로딩되기 전에 높이가 0이 되었다가 로딩 후 급격히 커지며 스크롤 위치가 점프하는 현상.
3. **스크롤바 접근성 저하**:
   - 스크롤바가 숨겨져 있어서 페이지 하단에 더 많은 내용이 있음을 직관적으로 알기 어려움.

## 해결 목표
- 데스크톱 화면의 모든 영역(좌우 빈 공간 포함)에서 스크롤이 작동하도록 수정.
- 이미지 로딩 전에도 컨테이너의 영역이 확보되도록 `aspect-ratio` 또는 `min-height` 설정.
- 스크롤바가 항상 보이도록 설정하여 사용자 경험 개선.
