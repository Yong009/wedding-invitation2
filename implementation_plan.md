# 썸네일 및 정보 업데이트 구현 계획

사용자가 제공한 참고 이미지(@i_022cd3251610.jpg)의 디자인과 정보를 바탕으로 모바일 청첩장의 썸네일(공유 미리보기)과 내부 콘텐츠를 업데이트합니다.

## 1. 주요 변경 사항

### A. 메타 데이터 (Open Graph Tags) 업데이트
- 카카오톡, 페이스북 등에서 링크 공유 시 표시되는 정보를 수정합니다.
- `og:title`: 강희택 ♡ 최유진의 결혼식에 초대합니다.
- `og:description`: 2026년 05월 16일, 토요일 오후 2시 30분 / 디엘웨딩홀 3층/아모르홀
- `og:image`: 새롭게 제작할 썸네일 이미지 (`public/thumbnail.jpg`)

### B. 썸네일 이미지 제작
- 참고 이미지의 레이아웃을 반영한 고품질 썸네일 이미지를 생성합니다.
- 디자인 요소: 화이트 배경의 카드 스타일, 세로형 커플 사진, 한글 이름 및 예식 정보 포함.

### C. 앱 콘텐츠 정보 수정 (`src/App.jsx`)
- 기존의 임시 데이터(김현수, 박지영 등)를 실제 정보로 교체합니다.
- 이름: 강희택, 최유진
- 날짜: 2026년 05월 16일 오후 2시 30분
- 장소: 디엘웨딩홀 3층 아모르홀
- 지도 좌표 및 정보 업데이트.

## 2. 작업 단계

1. **썸네일 이미지 생성**: `generate_image` 도구를 사용하여 참고 이미지와 유사한 느낌의 `thumbnail.jpg`를 생성하고 `public` 폴더에 저장합니다.
2. **`index.html` 수정**: `<head>` 섹션에 OG 태그를 추가/수정합니다.
3. **`src/App.jsx` 수정**: 
    - `Intro` 섹션의 이름, 날짜, 장소 수정.
    - `Greeting` 섹션의 혼주 정보 및 이름 수정 (기본값 유지 또는 사용자 요청 시 수정).
    - `Calendar` 섹션의 날짜(5월 16일) 및 D-Day 로직 확인.
    - `Location` 섹션의 장소명 및 주소 수정.
    - `Kakao Map` 좌표 수정 (필요 시).
    - `Account` 섹션의 성함 수정.

## 3. 예상 결과물
- 링크 공유 시 참고 이미지와 동일한 느낌의 세련된 미리보기가 표시됩니다.
- 웹사이트 접속 시 실제 예비 부부의 정보가 정확하게 노출됩니다.
